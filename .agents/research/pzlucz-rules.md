**40m/20m Round target details by division (§2.3.1.10.5):**

- Compound: 80cm at 40m, 60cm at 20m

- Short distances: 3 or 6 arrows (6 mandatory at Polish Championships except U15/U12)

**End structure:** 20 ends × 3 arrows (all categories).

### 1.4 Field Archery (§3.1, §3.4)

- 24 targets per round, 3 arrows per target = 72 arrows
- Target faces: field faces in 20cm, 40cm, 60cm, 80cm (scoring 1–6)
- Marked distances: specified per target size
- Unmarked distances: ranges differ per peg color:
  - **Red pegs** (R, C): longest ranges
  - **Blue pegs** (B, U18 R/C): medium ranges
  - **Yellow pegs** (U15 B): shortest ranges
- Championship round: 2 × 24 targets qualification (1 unmarked + 1 marked), top 16 elimination

**Field elimination (§3.4):**

- 1/8 and 1/4 finals: 6 marked targets
- 1/2 and finals: 4 marked targets
- Set system (R/B): 3 arrows per set, first to 6 set points
- Cumulative (C): 3 ends × 3 arrows, highest total wins

### 1.5 3D Archery (§3.2, §3.4)

- 24 targets (3D animal figures), 2 arrows per target in qualification = 48 arrows
- 1 arrow per target in elimination
- Unmarked distances only:
  - Red peg (R, C): max 45m, min 5m
  - Blue peg (B, T, L): max 30m, min 5m
- Championship round: 2 × 24 targets qualification, top 16 elimination (bracket §4.2.5)

**3D scoring zones:**

- 11 = small center
- 10 = large center
- 8 = vital zone
- 5 = body hit
- M = miss (also hooves, antlers)

### 1.6 Para-Archery (§2.10)

- Categories: R OPEN (M/W), C OPEN (M/W), W1 (M/W), VI1, VI2/3
- Same rounds as able-bodied except:
  - W1: 50m Round with full 80cm target; max draw weight 45 lbs; no peep sight, no magnifying lens, no level
  - VI: Special rounds at 30m (outdoor) and 18m (indoor) with 60/80/122cm targets; tactile sights only
- Teams: "Doubles" (2 athletes) instead of traditional 3-member teams
- Mixed teams: same structure as able-bodied
- Time: +10 seconds per arrow for all para categories

## 4. Scoring Rules

### 4.1 Target Faces and Scoring Zones (§2.1.2)

**Outdoor target faces:**

| Face   | Diameter | Rings    | Used at                                        |
| ------ | -------- | -------- | ---------------------------------------------- |
| 122cm  | 122cm    | 1–10 + X | Long distances (60–90m) for R; 50m for B       |
| 80cm   | 80cm     | 1–10 + X | Short distances; 50m for C (6-ring variant)    |
| 80cm/6 | 80cm     | 5–10 + X | Compound matchplay and short-distance compound |
| 60cm   | 60cm     | 1–10 + X | U15 compound at 20m                            |

**Indoor target faces:**

| Face        | Diameter | Rings    | Used at                                        |
| ----------- | -------- | -------- | ---------------------------------------------- |
| Triple 40cm | 3×40cm   | 6–10 + X | Senior/U21/U24 R and C (vertical layout)       |
| Full 40cm   | 40cm     | 1–10 + X | U18 R/C; all B categories; U18 R/C elimination |
| 60cm        | 60cm     | 1–10 + X | U15 all divisions                              |
| 80cm        | 80cm     | 1–10 + X | U12 at 15m                                     |

**Compound-specific inner 10:**

- Outdoor: X-ring (inner 10) for tiebreaking
- Indoor: Inner 10 diameter = 2cm (compound); standard 10 = 4cm diameter (recurve)

**Field faces (§3.1):** Scoring 1–6 (6 = center bull). Sizes: 20cm, 40cm, 60cm, 80cm.

**3D figures (§3.2):** Scoring 11 (small center), 10 (large center), 8 (vital zone), 5 (body), M (miss/antlers/hooves).

### 4.2 Tiebreaking (§2.6.4)

**Outdoor target qualification tiebreaking (§2.6.4.1.1):**

1. Greater number of 10s (including X)
2. Greater number of Xs
3. If still tied → same rank position; coin toss for bracket seeding

**Indoor qualification tiebreaking (§2.6.4.1.2):**

1. Greater number of 10s
2. Greater number of 9s
3. If still tied → same rank position; coin toss for bracket seeding

**Elimination cut-in tiebreaker / critical positions (§2.6.4.2):**

- Shoot-off with 1 arrow at last shot distance
- Decided by value; if equal → closest to center; if equal → repeat

**Team shoot-off (§2.6.4.2.3):**

- 3 arrows (1 per member), or 2 arrows for mixed team (1 per member)
- Decided by sum; if equal → closest single arrow to center; then 2nd, 3rd closest

**Field tiebreaking:** Most 6s, then most 5s, then most 4s…

**3D tiebreaking:** Most 11s, then most 10s, then most 8s…

## 6. Elimination & Finals

### 6.1 Outdoor Individual Elimination (§2.3.1.1)

**Bracket:** Top 104 archers qualify → seeded per bracket diagram §4.2.1

**Match format by division:**

| Division  | System     | Ends | Arrows/end | Win condition         |
| --------- | ---------- | ---- | ---------- | --------------------- |
| Recurve   | Set system | ≤5   | 3          | First to 6 set points |
| Barebow   | Set system | ≤5   | 3          | First to 6 set points |
| Compound  | Cumulative | 5    | 3          | Highest total after 5 |
| W1 (para) | Cumulative | 5    | 3          | Highest total after 5 |

**Set system detail:** Win set → 2 set pts; Draw → 1 pt each; Lose → 0 pts. First to 6 pts wins.

**Shoot-off (tied after all ends):**

- 1 arrow each, highest value wins
- If same value → closest to center wins
- If equal distance → repeat

**Elimination distances (outdoor):**

| Categories           | Div | Distance | Target        |
| -------------------- | --- | -------- | ------------- |
| M, W, U24M/W, U21M/W | R   | 70m      | 122cm         |
| U18M/W               | R   | 60m      | 122cm         |
| U15M/W               | R   | 40m      | 122cm         |
| All                  | C   | 50m      | 80cm (6-ring) |
| All                  | B   | 50m      | 122cm         |

**Note (§2.3.1.8):** At Polish Championships, U15 has NO elimination round.

### 6.2 Outdoor Team Elimination (§2.3.1.2)

**Bracket:** Top 24 teams → seeded per bracket §4.2.2

**Team composition (§2.3.1.2.3):**

- 3 or 4 members registered before qualification starts
- 3 shoot per end; 4th member substitutes between ends
- Fixed roster for entire competition

**Match format:**

| Division | System     | Ends | Arrows/end       | Win condition         |
| -------- | ---------- | ---- | ---------------- | --------------------- |
| R, B     | Set system | ≤4   | 6 (2 per archer) | First to 5 set pts    |
| C, W1    | Cumulative | 4    | 6 (2 per archer) | Highest total after 4 |

### 6.3 Mixed Team Elimination

- Best-ranked man + best-ranked woman from same club
- Max 3 mixed teams per club (Polish Championships only; no limit in U15)
- Bracket: top 16 mixed teams
- Match: Set system (R/B) with 4 arrows/end (2 per archer), best of 4 sets, first to 5; Cumulative (C/W1) with 4 ends × 4 arrows

### 6.4 Indoor Individual Elimination (§2.3.2.1)

**Bracket:** Top 32 archers → seeded per bracket §4.2.4

Same set/cumulative system as outdoor.

**Elimination target faces (indoor, §2.3.2.3–2.3.2.5):**

| Categories           | Div  | Distance | Target               |
| -------------------- | ---- | -------- | -------------------- |
| M, W, U24M/W, U21M/W | R, C | 18m      | Triple 40cm (linear) |
| U18M/W               | R, C | 18m      | Full 40cm            |
| All                  | B    | 18m      | Full 40cm            |
| U15M/W               | All  | 18m      | 60cm                 |

### 6.5 Indoor Team Elimination (§2.3.2.2)

**Bracket:** Top 16 teams → seeded per bracket §4.2.5

Same structure as outdoor teams (3 or 4 members, set/cumulative system).

### 6.6 Field/3D Elimination (§3.4)

**Bracket:** Top 16 individuals → seeded bracket

**Field elimination match:**

- 1/8 and 1/4: 6 marked targets, 3 arrows each
- 1/2 and Finals: 4 marked targets, 3 arrows each
- R/B: Set system (first to 6 set pts, 3 arrows per set)
- C: Cumulative (3 ends × 3 arrows)

**3D elimination match:**

- 1/8 and 1/4: 6 targets, 1 arrow each
- 1/2 and Finals: 4 targets, 1 arrow each
- Same set/cumulative rules

### 6.7 Finals Format (§2.5.3)

**Semi-finals and medal matches** are shot on a separate finals field with:

- Alternate shooting (one archer at a time)
- 20 seconds per arrow individual; 2 minutes for 6 team arrows
- Individual athletes have separate targets (no sharing)
- Agents (not athletes) retrieve arrows and verify scoring

### 6.8 Post-Elimination Classification (§2.6.5)

Losers within same round classified by:

1. Set points (for set system) or total points (for cumulative)
2. If tie → qualification ranking
3. If still tie → same rank position

## 7. Team Composition

### 7.1 Standard Teams (§2.3.1.2.3)

- **Size:** 3 members or 3+1 substitute
- **Requirement:** All from same club, same division, same age category
- **Roster lock:** Registered before qualification starts; fixed for entire competition
- **Substitution:** In 4-member team, can swap between ends (not within an end)
- **Scoring rule:** Best 3 of 4 qualification scores determine team ranking (confirmed by existing code `Poland-TeamsTop3of4` in `Rank/Obj_Rank_DivClassTeam_calc.php`)

### 7.2 Mixed Teams (§2.3.1.2.4)

- **Size:** 1 man + 1 woman from same club
- **Selection:** Best-ranked man + best-ranked woman form Team 1; 2nd best + 2nd best form Team 2; etc.
- **Limit:** Max 3 mixed teams per club at Polish Championships (no limit for U15)
- **Roster change:** Manager can change composition up to 30 minutes before elimination starts
- **Medals:** Awarded to the pair that actually competes in elimination

### 7.3 Field/3D Teams (§3.4)

**Field team ranking:** Best R + Best C + Best B from same club (sum of qualification scores)

**3D team ranking:** Best C + Best B + Best T + Best L from same club

## 8. Rankings & Classifications

### 8.1 Qualification Ranking

- Ranked by total score
- Tiebreaking per §4.2 (see Section 4.2 above)
- Teams ranked by sum of 3 best individual scores (or 2 for mixed teams)

### 8.2 Records (§2.11)

Records are tracked per category for many round types. Key points:

- Minimum 3 judges required (at least 1 Class-I or National)
- Must be at WA-ranked competition
- Application within 10 days with scorecards
- Field/3D have NO records (§3.14)

**Record categories include:**

- Individual rounds: 70m, 60m, 50m, 1440, double rounds, half rounds, 18m, 40m/20m
- Team rounds: same round types + team/mixed team variants
- Special shootings: rooster, sunflower, clover, poppy (R only, at Polish Championships)

### 8.3 Special Shootings (§2.12)

Unique PZŁucz tradition at Polish Championships for Recurve individual medalists:

- Distance: 15m
- 5 ends × 3 arrows
- Special 32cm-diameter target (placed on 80cm face, fields 7–10), with 2cm white center + 1mm cross
- Scoring: count hits in 32cm zone; tiebreak by sum of distances from center
- Themed targets by age:
  - Senior → Rooster (kur)
  - Junior (U21) → Sunflower (słonecznik)
  - Junior younger (U18) → Clover (koniczyna)
  - Cadet (U15) → Poppy (mak)

## 11. Gaps & Custom Needs

### 11.1 Non-Standard Bracket Sizes

PZŁucz uses a **104-draw individual bracket** (outdoor) which is not a standard power-of-2 bracket. This requires custom bracket generation or adaptation.

### 11.2 U24 and U15/U12 Age Classes

ianseo has no built-in concept of U24 (Młodzieżowcy) or U15/U12 (Młodzicy/Dzieci). These must be created as custom classes.

### 11.3 Split-Distance Rounds

The 40m/20m and 25m/15m rounds involve two different distances with different target faces in a single qualification. ianseo may need custom `CreateDistanceNew()` calls or a double-distance round setup.

### 11.4 Special Shootings (§2.12)

Completely unique to PZŁucz. Would require a custom module with:

- Custom target face (32cm on 80cm, themed art)
- Custom scoring (hits-in-zone + distance-from-center tiebreak)
- Integration with medal winners from main competition

### 11.5 Para-Archery Doubles

PZŁucz uses 2-member "doubles" teams for para-archery instead of standard 3-member teams. This affects team scoring calculations.

### 11.6 Field/3D Module

ianseo has basic field (`TGT_FIELD`) and 3D (`TGT_3D`) target type constants, but full field/3D competition management (variable-distance courses, unmarked distance rounds, peg-color routing) would require significant custom development.

### 11.7 Traditional and Longbow Divisions

Only used in 3D archery. Not present in standard ianseo division sets. Must be created via `CreateDivision()` in 3D-specific setup scripts.

### 11.8 Children's Round (U12)

4 × 18 arrows at 4 distances is not a standard ianseo round. Requires custom setup.

### 11.9 Indoor Compound Inner-10

The compound inner-10 (2cm diameter vs recurve's 4cm) affects scoring and tiebreaking. ianseo may already handle this via target face configuration, but needs verification.

### 11.10 Best 3 of 4 Team Scoring

Already partially implemented (`Poland-TeamsTop3of4` in `Rank/Obj_Rank_DivClassTeam_calc.php`). Needs verification that it covers all scenarios (outdoor, indoor, field).

### 11.11 Mixed Team Formation Rules

The automatic matching (best M + best W, 2nd M + 2nd W, etc.) and the 3-team-per-club limit require custom logic beyond standard ianseo team management.
